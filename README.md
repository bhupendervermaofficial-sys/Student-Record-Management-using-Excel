# 📊 Student Record (Excel Project)

## 📌 Project Overview

This is a basic Student Record project created in Microsoft Excel to manage and analyze students' marks.

## ✨ Features

- Student Name & Contact Details
- Subject-wise Marks (Hindi, English, Maths)
- Total Marks Calculation
- Maximum & Minimum Marks
- Average Marks
- Percentage Calculation
- Pass/Fail Result
- Grade Calculation
- Simple Charts for Marks Comparison

## 🛠️ Tools Used

- Microsoft Excel
- Excel Formulas
- Charts
- Table Formatting
